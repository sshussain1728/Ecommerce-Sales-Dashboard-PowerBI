# Ecommerce Sales Dashboard using Power BI

## Project Overview

This project analyzes e-commerce sales data using Power BI.

The dashboard provides business insights into:

- Total Sales
- Total Profit
- Orders
- Customer Analysis
- Regional Performance
- Category-wise Sales
- Monthly Trends
